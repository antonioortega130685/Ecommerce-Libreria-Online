Tienda Online de Libros en C#

Desarollada en Visual Studio Community 2017

Utilizando:

.NET Framework
Entity Framework Code First
SQL Server 2014
HTML5 & CSS3, Javascript, Jquery y Bootstrap
JSON
SUBLIME TEXT

La aplicación permite utilizar y administrar un sitio online de ventas de libros, pero puede extenderse a un e-commerce donde se puede comercializar cualquier tipo de producto mediante sencillas modificaciones de clases. 
La aplicación permite registro de cualquier usuario, restringiendo las funciones de actualización de contenido y stock (CRUD) a uno o más usuarios habilitados (administradores -se designan mediante Visual Studio y SQL Server, no se encuentra habilitada la opción desde la app-). 
Se exige registro antes de concretar la venta, mas se puede navegar por el sitio sin necesidad de llegar a ésta instancia.
